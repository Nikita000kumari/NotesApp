# 📝 Notes App 


A modern Notes App built using **Kotlin, Jetpack Compose, and Room Database**.

This project was developed as a **practice project** to learn Android development using modern tools and architecture.

---

## 🚀 Features

* ✍️ Add notes
* ❌ Delete notes
* 🔍 Search notes
* 💾 Persistent storage using Room Database
* 🌙 Dark mode support
